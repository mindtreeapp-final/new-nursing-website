## üé• YouTube Video ID Placeholder

**Where to add video IDs** (Around line 180-200 in each file):

```html
<!-- Replace YOUR_VIDEO_ID_1, YOUR_VIDEO_ID_2, YOUR_VIDEO_ID_3 -->
<iframe src="https://www.youtube.com/embed/ABC123XYZ"></iframe>
```

**Get Video ID from:**
- Regular video: `youtube.com/watch?v=ABC123XYZ` ‚Üí Use `ABC123XYZ`
- Shorts: `youtube.com/shorts/ABC123XYZ` ‚Üí Use `ABC123XYZ`

In VSCode terminal or command prompt:

```bash

cp oet.html iqn.html
cp oet.html osce.html
cp oet.html therapeutic.html
cp oet.html interview.html
```

Then edit each file according to this guide!
